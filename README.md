# Astra Chess Engine

Astra is a UCI-compliant chess engine written in C++. "Per aspera ad Astra."

## Compiling
Compile the code yourself for better performance.
```
git clone https://github.com/h1me01/Astra.git
cd Astra/src
make pgo
```
